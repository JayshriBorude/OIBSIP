# 📱 Google Play Store Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on the Google Play Store dataset and analyzes user reviews using TextBlob sentiment analysis.

## Objectives

- Clean real-world app data
- Analyze app categories
- Explore ratings and installs
- Study pricing trends
- Estimate revenue
- Perform sentiment analysis
- Generate business insights

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- TextBlob
- Jupyter Notebook

## Dataset

- googleplaystore.csv
- googleplaystore_user_reviews.csv

## Analysis Performed

- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Category Analysis
- Ratings Analysis
- Size vs Installs
- Pricing Analysis
- Revenue Estimation
- Sentiment Analysis
- Sentiment by Category

## Visualizations

- Category Distribution
- Ratings Distribution
- Size vs Installs
- Free vs Paid Apps
- Price Distribution
- Revenue by Category
- Sentiment Distribution
- Sentiment by Category

## Conclusion

The project demonstrates how exploratory data analysis and sentiment analysis can provide valuable insights into the Google Play Store ecosystem.
