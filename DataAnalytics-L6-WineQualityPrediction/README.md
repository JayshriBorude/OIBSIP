# 🍷 Wine Quality Prediction using Machine Learning

## 📌 Project Overview

This project predicts the quality of red wine using Machine Learning classification algorithms. The prediction is based on physicochemical properties such as acidity, alcohol content, pH, density, sulphates, chlorides, and residual sugar.

The project compares three classification algorithms and evaluates their performance using multiple evaluation metrics.

This project was completed as part of the **Oasis Infobyte Data Analytics Internship (OIBSIP) – Level 2 Task 2**.

---

## 🎯 Objective

To develop and compare multiple machine learning models capable of predicting whether a wine is of **Good** or **Bad** quality based on its chemical properties.
