# 🧹 Data Cleaning and Preprocessing

## 📌 Objective

The objective of this project is to clean a deliberately messy employee dataset and transform it into a clean, analysis-ready dataset using Python and Pandas.

---

## 📂 Dataset

Messy Employee Dataset

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🔍 Data Cleaning Tasks

- Data Quality Report
- Missing Value Analysis
- Missing Value Handling
- Duplicate Record Removal
- Data Standardization
- Data Type Correction
- Outlier Detection using IQR
- Before vs After Comparison
- Export Cleaned Dataset

---

## 📊 Techniques Used

- Mean Imputation
- Median Imputation
- Duplicate Removal
- String Standardization
- Datetime Conversion
- IQR Outlier Detection
- CSV Export
