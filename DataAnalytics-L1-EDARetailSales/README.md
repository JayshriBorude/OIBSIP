# 🛍️ Exploratory Data Analysis on Retail Sales Dataset

## 📌 Project Overview

This project was completed as part of the Oasis Infobyte Data Analytics Internship.

The objective of this project is to perform Exploratory Data Analysis (EDA) on a retail sales dataset to identify customer purchasing behavior, sales trends, and business insights using Python.

---

## 🎯 Objectives

- Understand the structure of the dataset
- Perform data cleaning and preprocessing
- Generate descriptive statistics
- Analyze customer demographics
- Analyze monthly and quarterly sales trends
- Identify top-performing product categories
- Generate business insights through visualizations

---

## 🗂 Dataset Information

The dataset contains retail transaction details including:

- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

Total Records: **1000**

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Exploratory Data Analysis

The project includes:

- Dataset Overview
- Missing Value Analysis
- Duplicate Check
- Descriptive Statistics
- Feature Engineering
- Sales Distribution
- Customer Age Distribution
- Gender Distribution
- Product Category Analysis
- Monthly Sales Trend
- Quarterly Sales Trend
- Revenue Analysis
- Correlation Heatmap

---

## 💡 Key Business Insights

- Customer purchasing behavior varies across product categories.
- Monthly sales trends reveal seasonal demand patterns.
- Customer demographics influence purchasing decisions.
- Certain product categories generate significantly higher revenue.
- Data visualization supports effective business decision-making.

---

## 📈 Business Recommendations

- Increase inventory for high-performing categories.
- Target marketing campaigns based on customer demographics.
- Optimize seasonal promotional strategies.
- Improve sales for lower-performing product categories.

---

## ✅ Conclusion

This project demonstrates how Exploratory Data Analysis (EDA) can transform raw retail data into meaningful business insights, enabling organizations to make data-driven decisions.
