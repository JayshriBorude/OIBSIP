# ✍️ Autocomplete and Autocorrect using NLP

## 📌 Overview

This project implements **Autocomplete** and **Autocorrect** systems using Natural Language Processing (NLP).

Autocomplete predicts the next word using a frequency-based Bigram model, while Autocorrect fixes misspelled words using PySpellChecker.

This project was completed as part of the **Oasis Infobyte Data Analytics Internship – Task 5**.

---

## 🎯 Objectives

- Perform NLP preprocessing.
- Build a Bigram autocomplete model.
- Implement an autocorrect system.
- Evaluate prediction performance.
- Compare autocomplete and autocorrect algorithms.
