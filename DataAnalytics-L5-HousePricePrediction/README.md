# 🏡 House Price Prediction using Linear Regression

## 📌 Project Overview

This project focuses on predicting house prices using **Linear Regression**, a supervised machine learning algorithm. The model is trained on housing data containing property features such as bedrooms, bathrooms, living area, location, and construction year to estimate house prices accurately.

This project was completed as part of the **Oasis Infobyte Data Analytics Internship – Level 2 Task 1**.

---

## 🎯 Objective

The objective of this project is to build and evaluate a Linear Regression model capable of predicting house prices based on various property characteristics. The project covers the complete machine learning workflow, including data preprocessing, feature engineering, model training, evaluation, and interpretation.
